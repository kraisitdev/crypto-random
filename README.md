## ⚙ Get Started

- 1.Clone project

  ```bash
  git clone git@github.com:kraisitdev/crypto-random.git
  ```

- 2.Go to project folder

  ```bash
  cd crypto-random
  ```

- 3.Set up environment by command

  ```bash
  dotnet restore
  ```

- 4.Run project by command

  ```bash
  dotnet run
  ```
